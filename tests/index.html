JS-CRUD-API Tests

Functional tests for the JS-CRUD-API library

Browser Tests API (example)

Quick start

  1. Sync tests: npm run test:sync
  2. Build tests: npm run test:build
  3. Initialize DB: npm run test:init:sqlite
  4. Start PHP: php -S localhost:8081 -t tests
  5. Open: http://localhost:8081/browser/
